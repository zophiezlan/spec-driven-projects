# /nuaa.report (PLANNED)

Generate progress or final evaluation reports drawing from the impact framework indicators and collected datasets.

## Purpose

- Summarize process, outputs, outcomes, and impact
- Produce funder-facing and community-facing versions
- Include visuals and plain-language summaries

## Inputs (planned)

- Program name
- Reporting period (e.g., Mid-program, Final)
- Data sources (CSV paths, survey exports)
- Audience (--format=funder|community|internal)

## Outputs (planned)

- Markdown report with sections
- Optional Word export (future M365 integration)
- Visuals (tables/graphs) where available

## Roadmap

- Define data ingestion contract
- Chart generation hooks
- Template variants per audience
